Command to Update the Poker

        > git stash save
        > git pull
        > pm2 reload all

        Open Google Chrome with the link http://localhost:1337
        Right Click any where on the page and click on "Inspect".
        Right Click on the Refresh button and click on "Empty Cache and Hard Reload"

Command to Start/Restart Poker 

        > pm2 reload all
        
        Open Google Chrome with the link http://localhost:1337
